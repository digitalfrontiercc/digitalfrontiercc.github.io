エッセイ：

  • Marc Garrett on DIWO - DIWO（Do-It-With-Others）：今日の群衆におけるピアのエンパワーメントのための分散的手法としての芸術的共創。DIWOの実践は、異なる出所の要素が豊かに混ざり合いクロスオーバーしてハイブリッドな体験を構築するような開かれた空間を可能にする。これはアーティストとキュレーターの権力役割に挑戦し再交渉する。

  • The User Condition - エッセイ：コンピュータ利用者が自動化から抜け出す能力として定義されるエージェンシーを得るための条件とは何か？利便性の名の下にその論理が不透明にされているとき、他者によってプログラムされたソフトウェアで構成された世界の中でユーザーの自律の地平はどこにあるのか？



フィロソフィー：

  • permacomputing - Permacomputingは概念であり、パーマカルチャーに触発されたコンピュータおよびネットワーク技術におけるレジリエンスと再生性の問題に向き合う実践コミュニティである。現在のコンピューティングには巨大な環境的・社会的問題があり、パーマコンピューティングは産業的農業に対するパーマカルチャーの挑戦と同様にそれらに挑みたいと考える。パーマコンピューティングは反資本主義的な政治プロジェクトであり、アナキズム、脱植民地主義、インターセクショナル・フェミニズム、ポストマルクス主義、脱成長、エコロギズムなどの複数の潮流に動機づけられている。



ツール＆フレームワーク：

  • we take note - コラボレーションとコミュニティ構築のための豊富なツールライブラリ。創出、維持、インパクト測定のためのツールを紹介し、金銭で測れないポジティブな影響を持つ運営を行う組織にとって不可欠なリソースを提供する。 collaboration guidebook は多数あるうちの一例。



情報とリスト：

  • varia digital solidarity networks - デジタル連帯、共 conviviality（共生性）、連帯のためのツール、実践、読書の共有リスト。

  • Civic Tech Field Guide - Civic Tech Field Guideは、民主主義のために技術を用いるプロジェクト、ツール、組織の世界で最も充実したコレクションである。主にクラウドソースのディレクトリで、役立つリソースやインスピレーションを見つけ、自分の取り組みを共有できる。研究や分野の状況に関する定期的な更新も公開している。



Monoskop：

Monoskop は2004年に創設された芸術、文化、人文学のための独立したウェブベースの教育資源・研究プラットフォームである。Monoskopはウィキページを通じて、Fediverse、テクノフェミニズム、脱殖民的美学、人新世、コミュニティラジオ、フリーソフトウェア、アーティストによる出版、パフォーマンス、サウンドアート、実験映画・ビデオ など、現代のテーマや運動に関する多言語の系譜的書誌を提供する。また多数の書誌項目はMonoskopや他のフリー＆リブレなライブラリで公開されている電子版へのリンクを含む。（2023年）

  • Community Servers List - 相互依存的、フェミニスト／トランス*フェミニスト、フリー／リブレ、自ホスティング、実験的、フェデレーテッド（連合型）、ラディカル、自律的、コレクティブ、協同組合的、コミュニティ／アート向けサーバーのリスト。
